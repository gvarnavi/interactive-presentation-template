Jupyterlab cells

We can also use the jupyter-react library to connect and interact with a Jupyter server.

Jupyter Cell fallback content for prerendering.

Here, we're connecting to a remotely-hosted Jupyter server by datalayer, but we can also point this to a local Jupyter server, e.g. using:

<JupyterCell
  source= {`import numpy as np...`}
  serverHttpUrl='http://localhost:8686/api/jupyter'
  serverWsUrl='ws://localhost:8686/api/jupyter'
  token='60c1661cc408f978c309d04157af55c9588ff9557c9380e4fb50785750703da6'
/>
