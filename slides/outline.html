Markdown preformatted code blocks with the python language identifier are automatically converted to runnable and editable code blocks thanks to react-py. See the react-py repo and documentation for more info.

For example, in a Markdown docs file:

```python
print("Hello, World!")
```

Becomes:

print("Hello, World!")

For this configuration, this only loads the python stdlib and numpy by default. If you want to install additional packages you can use a comma-separated list as so:

```python packages="scipy python-cowsay"
from scipy import special
from cowsay import cowsay

res = special.cbrt(27)
message = f"The cube root of 27 is {res}"
print(cowsay(message))
```

Markdown preformatted code blocks with the jsx live language identifier are automatically converted to runnable and editable code blocks thanks to @docusaurus/theme-live-codeblock.

```jsx live
function ReptileListItems() {
  const reptiles = ["alligator", "snake", "lizard"];
  return reptiles.map((reptile) => <li key={reptile}>{reptile}</li>);
}
```
